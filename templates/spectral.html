Spectral Clustering

A clustering technique which uses the laplacian of the similarity matrix to assign clusters.

Algorithm

Create a similarity matrix of the observations. From the similarity matrix, compute the Laplacian Matrix. Using the Laplacian Matrix, assign clusters to each observation using K-Means clustering.

Model Parameters

  • Similarity Criterion: Gaussian RBF Kernal
  • Labeling Technique: K-Means
  • Number of Iterations(for K Means):10
  • Number of Clusters:

Notes

Spectral Clustering is useful for cases where the clusters have irregular shapes and do not congregate uniformly around some centroid. Observations are clustered according to closeness to each other as opposed to closeness to some common point. Similarity criterion can also be tuned according to context and desire.
